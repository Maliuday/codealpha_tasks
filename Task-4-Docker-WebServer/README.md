# Task 4 - Docker Web Server

## Objective

Deploy a simple web application inside a Docker container on an AWS EC2 instance.

## Tools Used

* AWS EC2
* Amazon Linux 2023
* Docker
* HTML

## Steps Performed

1. Launched an EC2 instance.
2. Installed Docker on the EC2 instance.
3. Started and enabled Docker service.
4. Created a simple HTML webpage.
5. Created a Dockerfile.
6. Built a Docker image.
7. Ran a Docker container.
8. Verified the webpage through browser.

## Result

Successfully deployed a web application using Docker on AWS EC2.
